DLL Export Finder tool for windows

================================================== Getting Started ======================================================
1. This software works if the system has visual studio installed (Dumpbin utility is required).
2. Copy dumpbin to the same folder.
3. Execute the exe by passing the source dll folder as command line argument. (This code supports a maximum of 256 files).
4. The dependence file generated by dumpbin utility will be stored in the same folder of dlls.

Running the application

1. Copy dumpbin utility to the same folder of ExprtsFinder.exe
2. Type ExportsFinder command followed by the source folder (Eg. ExportsFinder.exe z:\Dhyan\ToTest\sample\)
